# MOUVIZ

Plateforme de suivi et découverte de films - Stack Nuxt 4 + SQLite + Drizzle.


## Installation

```bash
pnpm install
```

## Développement

```bash
pnpm run dev
```

## Production

```bash
pnpm run build
pnpm run preview
```

## Base de données

```bash
pnpm run db:generate
pnpm run db:migrate
pnpm run db:seed
pnpm run db:studio
```
